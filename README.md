# 配送管理系统

一个基于飞书多维表格的配送管理系统，支持团队协作、登录认证。

## ✨ 功能特性

- ✅ 用户登录认证（多用户支持）
- ✅ 配送记录自动生成
- ✅ 已吃餐数自动计算
- ✅ 甘特图状态更新
- ✅ 团队协作支持
- ✅ 随时随地访问（7x24小时）
- ✅ 飞书工作台集成

## 🚀 快速开始

### 团队成员使用

1. 打开飞书工作台
2. 找到"配送管理系统"
3. 输入账号密码登录
4. 开始使用
